## 🫮 Calculadora Básica em C

Este é um programa simples de **calculadora em linguagem C**, desenvolvido para praticar os conceitos básicos de programação, como uso de `condicionais`, `laços`, `switch case`, e `entrada/saída de dados`.

Com esta calculadora, você pode realizar as quatro operações matemáticas básicas:

- ✅ Soma  
- ✅ Subtração  
- ✅ Multiplicação  
- ✅ Divisão  

---

## 🚀 Como funciona?

O programa exibe um menu no terminal com as opções de operação. O usuário escolhe uma delas, insere dois números, e o resultado é calculado e exibido. Após isso, o usuário pode escolher se deseja realizar outra operação ou encerrar o programa.

---

## 📝 Observações

- O código limpa a tela com `system("cls")`, que funciona no Windows.  
  Se estiver em Linux ou macOS, substitua por `system("clear")`.
- Os valores são lidos como `float`, e os resultados são formatados para melhor visualização.

---

## 🧐 Objetivo

Este projeto foi criado como um exercício de aprendizado. O objetivo é treinar:
- Estrutura de controle (`if`, `do-while`, `switch`)
- Entrada e saída de dados com `scanf` e `printf`
- Operações matemáticas simples
